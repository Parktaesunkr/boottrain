# boottrain

# 프로젝트 목적
부트스트랩을 이용한 게시판 구현 및 댓글 기능 구현
댓글 기능에 수정 삭제 기능 추가

Spring을 이용한 CRUD와 RESTful 학습

# 구현 기술 스택 
- Spring Boot
- MariaDB
- JPA
- JDBC
- Lombok
- Spring Web
- Java, HTML, CSS, JS
- gradle
- Bootstrap
